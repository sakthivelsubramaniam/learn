
# Azure Synapse 

## overview

1. Spark - used in big data
2. Log and time series analytics.
3. Pipelines for ETL/ELT
4. Integration with Reporting/Querying tech like Power BI, Azure ML etc

### features

- Synapse Sql:  Distributed sql 
- Apache Spark for Synapse: used for data preparation, data engineering, ETL and ML.
- Data Explorer: Used to provide log and telemetery data
- Data lake integration.
- Azure data factory engine build into it.ETL pipeline can be creaed without leaving spark

Commit Formatting
--pretty[= <format>]  
--format=(format)
  format = oneline | short | medium| full| fuller| reference| email| raw  | Custom String

Customer String can be one of the following   

| commnad | description 
|---|--|
| %n | new line |
| %% | raw |
| %x00 | print byte in hex decimal |
| %C(…​) | print in color |
| %Cred | in red colur |
| %H / %h | commit hash |
| %T | tree hash |
| %P | parent hash |
| %an / %aN /  | authoer name |
| %ad  / %aD | authoer date |
| %as / auther short date format |
| %cn | commiter name |
| %N | commit notes |
